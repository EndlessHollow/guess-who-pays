# Guess who pays

Project pick unfortunate person which has to pay for others people lunch

## How To Run

```sh
// install packages
npm i

// starts the development server
npm run dev
```

### Available Scripts:

```sh
// run a development server
npm run dev

// prepare a preview of a build
npm run preview

// create a build of the project
npm run build

// format the project
npm run prettier

// lint the project
npm run lint
```
